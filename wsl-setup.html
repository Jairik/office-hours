WSL Setup Guide

Introduction

Welcome to the WSL setup guide. Follow the steps below to download & install WSL to access the Linux kernel (Ubuntu) from your Windows machine.

Steps

Follow the following steps, in order.

1) Open Powershell as Administrator

Either: ⊞ + r -> type powershell OR ⊞ -> "powershell" -> left click -> open as administrator.

2) Run the install command

In the Powershell window, type the following command and hit enter:

wsl --install

This will download and install WSL, the Linux kernel, and the Ubuntu distribution, assuming it has administrator privaledges

2.5) Optional but Recommended: Turn on Virtual Machine Platform and WSL in Settings

Although the above command should change these settigns after rebooting, it doesn't hurt do it manually as well.

We will be opening the "Turn Windows features on or off" settings. To do this, ⊞ -> "turn windows features on or off" -> Enter.

In this window, scroll down to "Virtual Machine Platform" and "Windows Subsystem for Linux". Ensure that these are checked, then click OK.

3) Restart your computer, as prompted

You should be prompted to restart your computer. This will allow for everything to be configured during reboot.

4) Open up Ubuntu Terminal + set up your Linux username and password

After your computer restarts, open up the terminal and type "ubuntu". You should be prompted to create a Linux username and password. This is separate from your Windows username and password. You will use this username and password when using the Ubuntu terminal.

Troubleshooting

Sometimes, WSL can just be a little weird installing. Some basic fixes may involve restarting computer, retrying the command as an administrator, and/or installing the Ubuntu app from the microsoft store.
